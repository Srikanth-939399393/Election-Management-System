# Election Management System (EMS)

A structured election administration system built using **Zoho Creator** to manage voter registration, candidate nominations, polling operations, vote counting, and result publication.

---

## Overview

The Election Management System (EMS) is designed to digitize and streamline the complete election lifecycle. It supports administrative users, election officers, candidates, and voters through role-based access and controlled workflows.

This system can be adapted for:
- Institutional elections
- University elections
- Corporate voting
- Organizational polls

---

## Features

### Voter Management
- Public voter registration form
- OTP verification
- Duplicate detection using unique ID
- Auto-generated Voter ID
- Status tracking

### Candidate Management
- Online nomination submission
- Document upload support
- Approval / rejection workflow
- Auto-generated Candidate ID

### Election Configuration
- Timeline setup
- Form auto-lock by date
- Ballot configuration
- Optional NOTA support

### Polling Booth Management
- Booth assignment
- Officer allocation
- GPS check-in
- Turnout monitoring

### Vote Counting & Results
- Booth-wise vote entry
- Automatic total calculation
- Winner declaration
- Results dashboard

### Complaint & Grievance System
- Public complaint submission
- Officer assignment
- Escalation workflow
- Status tracking

## Modules

| Module | Purpose |
|--------|---------|
| Election Setup | Configure election schedules |
| Constituency Management | Define zones and link records |
| Voter Registration | Manage voter applications |
| Candidate Management | Process nominations |
| Polling Booth | Assign and monitor booths |
| Officer Management | Manage officers |
| Vote Count | Record and validate results |
| Results Dashboard | Publish election outcomes |
| Complaints | Track and resolve issues |
| Audit Log | Monitor system actions |

---

## User Roles

- **Super Admin** – Full system control  
- **Election Officer** – Field operations & approvals  
- **Candidate** – View nomination & results  
- **Voter / Public** – Registration & status tracking  
- **Observer** – Read-only access
